# ALIGNMENT_backend

## 项目上下文
- 现有前端：Vue3 + Element Plus，Axios 统一返回格式 code/message/data
- 已有接口规范：API.md
- 后端技术栈：FastAPI + MySQL

## 任务目标
- 设计并实现完整的博客平台后端系统
- 与 API.md 完全对齐，支持前端已完成的登录/注册调用
- 覆盖数据库设计、认证授权、业务逻辑与接口实现

## 需求范围
- 用户体系：注册/登录/用户信息/偏好标签
- 文章体系：发布/编辑/删除/详情/列表/我的文章
- 互动体系：点赞/收藏/评论/评论列表
- 行为上报：阅读/滚动/搜索等
- 推荐接口：首页推荐、相似推荐（基础实现）
- 分析接口：阅读趋势/推荐效果/用户画像/内容表现（基础实现）

## 关键约束
- JWT 认证，token 有效期 24 小时，无刷新 token
- 注册/登录仅使用 email + password
- role 字段区分普通用户/博主与管理员，管理员权限更高
- 所有函数必须添加完整函数级注释
- 严格遵循 5S 文档管理规范与 6A 工作流

## 业务理解
- 普通用户与博主共用基础能力，均可发布/编辑自身文章
- 管理员可删除任意文章、查看平台级数据
- 推荐与分析接口先做可用的基础版本，保证接口契约一致

## 风险与处理
- 推荐/分析算法暂用基础统计实现，后续可替换为真实模型
- 数据迁移与生产部署使用 Alembic/容器化可作为后续任务

## 澄清与结论
- 需求已明确，进入共识与架构设计阶段
