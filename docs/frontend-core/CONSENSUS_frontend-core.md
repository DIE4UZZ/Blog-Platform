# CONSENSUS_frontend-core

## 明确需求描述
- 完成博客平台剩余核心前端页面与路由：推荐流、列表/搜索、详情、编辑、个人中心、数据分析。
- 严格按 `API.md` 调用接口并处理错误提示与 loading 状态。
- 视觉风格统一为 Apple 现代简约风格，响应式适配。

## 验收标准
- 各页面可通过路由访问并具备基本交互与数据加载。
- 文章相关接口完成对接：列表、详情、发布/编辑、点赞/收藏、评论、相似推荐。
- 用户相关接口完成对接：信息获取、偏好标签更新、我的文章列表。
- 数据分析接口完成对接：阅读趋势、推荐效果、用户画像、内容表现。
- 统一错误提示与加载状态管理。

## 技术实现方案与约束
- 继续使用 Vue3 + Tailwind + Element Plus + Axios。
- 采用服务层封装 API 调用，视图层负责 UI 与状态展示。
- 所有函数添加函数级注释（功能、参数、返回值与用途）。

## 任务边界
- 页面为可运行版本，不包含复杂富文本/Markdown 编辑器。
- 若后端缺少接口字段，前端按最小字段处理并显示占位。
